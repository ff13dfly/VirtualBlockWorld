# Module importer

## Function
* The basic component of VBW, which can import resource files in 3D format.

  

## Definition

### Version : 2020

* Data struct :`[ size(X,Y,Z) , position(X,Y,Z) , rotation(X,Y,Z) , scale(X,Y,Z) , resource ID , config(...) ]`

* Sample :`[[3,0.2,3],[12,6,1.5],[0,0,0],[1,1,1],13,[]]`
