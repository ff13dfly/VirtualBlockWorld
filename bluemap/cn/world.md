# 世界配置（World Setting）说明



## 世界配置的目的

通过调整参数，创造不同的虚拟世界，让虚块世界变得丰富多彩。



## 世界配置参数

世界的配置参数由治理委员会进行公开透明的管理，

| 参数名         | 默认值     | 说明             |
| -------------- | ---------- | ---------------- |
| start          | [1989,520] | 用户默认的出生点 |
| fee_block_sale | 1          | 土地交易费（%）  |
| fee_block_rent | 1          |                  |
